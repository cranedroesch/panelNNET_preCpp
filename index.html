panelNNET

Semiparametric panel data models using neural networks

panelNNET is an R package for semiparametric panel data models, with the nonparametric part of the model represented by a feed-forward neural network. It is suitable for building predictive models from longitudinal datasets, for high-dimensional regression adjustment, and for estimation of heterogeneous causal effects. Approximate causal inference on parametric terms via linear taylor expansion of the neural network is implemented.

This software is under active development. Collaboration is welcome. Watch for a working paper soon.
